# Лабораторная работа №1 - Вариант 20

## Разработка веб-приложения на Flask для обработки изображений

### Описание задания
Веб-приложение должно формировать новое изображение на основе исходного путем сдвига по замкнутым прямоугольным составляющим на определенное число пикселов, которое задает пользователь.

### Структура проекта
flaskapp/
├── static/ # Статические файлы (изображения)
├── templates/ # HTML шаблоны
├── uploads/ # Загруженные файлы
├── some_app.py # Основное приложение Flask
├── test_app.py # Тесты для GitHub Actions
├── client.py # Клиент для тестирования
├── requirements.txt # Зависимости Python
├── wsgi.py # WSGI для развертывания
├── runtime.txt # Версия Python для Heroku
├── Procfile # Конфигурация Heroku
└── README.md # Этот файл



### Установка и запуск

**Клонирование репозитория:**
```bash
git clone https://github.com/Platon-KH/flask-lab1.V2
cd flaskapp
Установка зависимостей:

pip install -r requirements.txt
Запуск приложения:
python some_app.py

Откройте в браузере: http://127.0.0.1:5000
